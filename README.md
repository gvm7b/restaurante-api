# Restaurante API

Repositório do sistema de restaurante com backend Spring Boot e frontend React/Vite.

## Estrutura

- `src/`: backend Java com Spring Boot.
- `frontend/`: frontend React com Vite.

## Backend

```bash
./mvnw spring-boot:run
```

No Windows:

```bash
mvnw.cmd spring-boot:run
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
