# Restaurante

Repositorio do sistema de restaurante com backend Spring Boot e frontend React/Vite.

## Estrutura

- `backend/`: API Java com Spring Boot.
- `frontend/`: aplicacao React com Vite.

## Backend

```bash
cd backend
./mvnw spring-boot:run
```

No Windows:

```bash
cd backend
mvnw.cmd spring-boot:run
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
